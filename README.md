# ChasseAuxMots

## Environnement (Windows)

- Installer Python 3.8+ si nécessaire.
- Créer un environnement virtuel et installer les dépendances (cmd) :

```
setup_env.bat
```

- Ou en PowerShell :

```
setup_env.ps1
```

- Après création, activer l'environnement (cmd) :

```
.venv\Scripts\activate
```

- En PowerShell :

```
. .venv\Scripts\Activate.ps1
```

Le fichier `requirements.txt` contient les dépendances du projet.
